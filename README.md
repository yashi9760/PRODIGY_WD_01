# Responsive Landing Page

A modern and responsive landing page developed using HTML, CSS, and JavaScript. The project features a fixed navigation bar with interactive hover effects, dynamic styling on scroll, smooth user experience, and a mobile-friendly responsive layout.

## Features
- 📱 Fully Responsive Design
- 🧭 Fixed Navigation Bar
- 🎨 Hover Effects on Menu Items
- 📜 Dynamic Navbar Styling on Scroll
- ⚡ Smooth User Experience
- 💻 Built with HTML5, CSS3 & JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
